# tekadventure


## Project
Epitech 1st year project.

This project consists of programming an adventure video game.




## Prerequisites
To run the project, you'll need to install :

* CSFML



## How to use it
To use this program, you first need to compile it using the makefile :
    
    make re && make clean

Then, just launch it :

    ./tekadventure
